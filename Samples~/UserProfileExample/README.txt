This example depends on a Unity Native Gallery Plugin.

To use this example, you need to import the Unity Native Gallery Plugin to your project.
You can add:
    "com.yasirkula.nativegallery": "https://github.com/yasirkula/UnityNativeGallery.git",
to your manifest.json.

You can also use other installation methods, here more information:
https://github.com/yasirkula/UnityNativeGallery
